# YuCv
YuCv is a simple Node-based utility that converts YouTube video URLs into MP3 files.  It utilizies Node JS, Express, Socket.io, and fluent-ffmpeg.

# Setup
1.  Clone the branch
2.  Install ffmpeg on your OS
3.  `npm install`
4.  `node start.js`
